Ben OS · Sandbox

The Arcade.

A corner of the OS for fun. Every game here is written from scratch in vanilla JavaScript on a single <canvas> — no engines, no libraries, no assets. Including DOOMBA: a DOOM-style first-person shooter built on a raycasting engine I wrote by hand.

> ben-os --launch arcade ▋

Honest scope: DOOMBA isn't the original id Software DOOM (that needs the copyrighted game data) — it's my own raycaster in ~500 lines of JS, built to learn how those engines actually work.
